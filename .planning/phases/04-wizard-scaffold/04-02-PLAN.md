    **Paso 2: Implementar DetectConfigStep en `cmd/jaimito/setup/steps.go`**

    `DetectConfigStep` struct:
    - `selectedOption int` — 0=Editar, 1=Crear desde cero, 2=Cancelar (para config valido) o 0=Crear desde cero, 1=Cancelar (para config invalido)
    - `options []string` — se determina en Init()
    - `done bool`
    - `isValid bool` — config existe y es valido
    - `isInvalid bool` — config existe pero invalido

    `DetectConfigStep.Init(data)`:
    - Si `data.ExistingCfg == nil && data.ConfigErr == nil` (o `!data.ConfigExists`):
      config no existe -> setear `data.Mode = "new"`, `s.done = true`, retornar nil
    - Si `data.ExistingCfg != nil`:
      config valido -> `s.isValid = true`, `s.options = []string{"Editar configuracion existente", "Crear desde cero", "Cancelar"}`
    - Si `data.ConfigErr != nil`:
      config invalido -> `s.isInvalid = true`, `s.options = []string{"Crear desde cero", "Cancelar"}`

    `DetectConfigStep.Update(msg, data)`:
    - `tea.KeyPressMsg` "up" o "k": mover selectedOption arriba (wrap)
    - `tea.KeyPressMsg` "down" o "j": mover selectedOption abajo (wrap)
    - `tea.KeyPressMsg` "enter": ejecutar la opcion seleccionada:
      - Config valido: Editar -> data.Mode="edit", Crear -> data.Mode="fresh" + backupConfig, Cancelar -> tea.Quit
      - Config invalido: Crear -> data.Mode="fresh" + backupConfig, Cancelar -> tea.Quit
    - Marcar s.done=true en editar/crear

    `DetectConfigStep.View(data)`:
    - Si config valido: titulo "Configuracion existente detectada" con TitleStyle, luego resumen:
      ```
      Bot token:  ****:klmnop
      Canales:    2 configurados
      Servidor:   127.0.0.1:8080
      Base datos: /var/lib/jaimito/jaimito.db
      ```
      Luego opciones con cursor `>` en la seleccionada
    - Si config invalido: titulo "Configuracion existente con errores" con ErrorStyle, luego el error especifico, luego opciones
    - Hint inferior: "↑/↓: mover │ Enter: seleccionar"

    `obfuscateToken(token string) string`:
    - Si len(token) <= 6: retornar "****"
    - Sino: retornar "****:" + token[len(token)-6:]

    `backupConfig(path string) error`:
    - Leer contenido de path
    - Escribir a path+".bak"
    - Retornar error si falla
