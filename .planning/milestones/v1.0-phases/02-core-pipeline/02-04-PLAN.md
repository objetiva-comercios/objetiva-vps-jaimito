Step 12: Start cleanup scheduler (renumbered from old step 10).

```go
// 12. Start cleanup scheduler — purges old messages in background goroutine.

```

Step 13: Build and start HTTP server.
```go
// 13. Start HTTP server — serves API endpoints for notification ingestion.
router := api.NewRouter(database, cfg)
server := &http.Server{
    Addr:         cfg.Server.Listen,
    Handler:      router,
    ReadTimeout:  15 * time.Second,
    WriteTimeout: 15 * time.Second,
    IdleTimeout:  60 * time.Second,
}
go func() {
    if err := server.ListenAndServe(); err != nil && !errors.Is(err, http.ErrServerClosed) {
        slog.Error("HTTP server failed", "error", err)
        os.Exit(1)
    }
}()
```
CRITICAL: `errors.Is(err, http.ErrServerClosed)` must be used — ListenAndServe returns this non-error on clean Shutdown.

Step 14: Log ready state (renumbered from old step 11).
```go
// 14. Log ready state — all components initialized successfully.
slog.Info("jaimito started",
    "addr", cfg.Server.Listen,
    "channels", len(cfg.Channels),
    "db", cfg.Database.Path,
)
```

Step 15: Wait for shutdown signal and graceful shutdown (replaces old steps 12).
```go
// 15. Wait for shutdown signal — graceful shutdown with 30s timeout.
<-ctx.Done()
slog.Info("shutting down")

shutdownCtx, cancel := context.WithTimeout(context.Background(), 30*time.Second)
defer cancel()
if err := server.Shutdown(shutdownCtx); err != nil {
    slog.Error("HTTP server shutdown error", "error", err)
}
```
Note: database.Close() is already deferred. The dispatcher and cleanup goroutines stop via ctx.Done(). Only the HTTP server needs explicit Shutdown().
